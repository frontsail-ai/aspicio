Aspicio is a free, open-source DXF viewer that runs entirely in your browser — drawings are parsed and rendered locally and never uploaded. Inspect layers, measure distances, and export to SVG or PNG.

The same engine ships as a TypeScript library with React, Vue, Svelte, and web-component bindings, plus an HTTP API and an MCP server for AI agents.
