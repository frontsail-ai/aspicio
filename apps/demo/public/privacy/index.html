Privacy Policy

Aspicio · effective July 21, 2026 · demo · terms

Aspicio is an open-source DXF viewer with a hosted API and MCP server, developed and maintained by FrontSail AI. This policy covers the hosted services (the demo site, the HTTP API, and the remote MCP endpoint) and the locally-installed tools. The short version: we don't want your data, we don't have accounts, and we don't keep your drawings.

What we collect

  • No personal data. There are no accounts, sign-ups, names, or email addresses. We run no analytics and set no cookies.
  • Drawings you submit. When you (or your AI assistant) send a DXF to the hosted API or remote MCP server — as a URL to fetch or as request content — it is processed in memory to produce the response (facts, an image, or viewer data) and is not stored.
  • Operational logs. Our infrastructure provider (Cloudflare) records standard request metadata — IP address, URL, timestamps — which we use for rate limiting and abuse prevention. These logs are short-lived and are not used for anything else.

Purposes

Exactly one: providing the service — parsing, describing, and rendering the drawings you send, and keeping the service available (rate limits, abuse prevention).

Recipients

  • Cloudflare operates the infrastructure the hosted services run on.
  • Your AI assistant's provider. When you use Aspicio through an AI assistant (ChatGPT, Claude, or any MCP client), the results we return — drawing facts, rendered images, viewer data — are relayed by your assistant to its model provider under that provider's terms. We do not send anything to model providers ourselves.
  • We do not sell or share data with anyone else.

Retention

Drawings: not retained — processed per-request, in memory. Operational logs: retained briefly per Cloudflare's standard log windows, then gone.

Your controls

  • Don't send sensitive drawings to the hosted services. For local files, use the local server (npx -y @aspicio/mcp) — it reads files in-process and never uploads them to any Aspicio service.
  • Everything is open source (github.com/frontsail-ai/aspicio), so you can verify the above or run the whole pipeline yourself.
  • Questions or requests: support@frontsail.ai or a GitHub issue.

Changes

If this policy changes, the new version is published here with an updated effective date; the history lives in the repository.
